# Taxi Malacrida - Site Web

## Project info

Site web pour Taxi Malacrida - Service de chauffeur privé Tesla à Aix-en-Provence.

## Technologies utilisées

Ce projet utilise :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation et développement

```sh
# Installer les dépendances
npm i

# Lancer le serveur de développement
npm run dev
```

## Déploiement

Le projet est déployé sur Vercel. Les modifications poussées sur la branche `main` déclenchent automatiquement un nouveau déploiement.
